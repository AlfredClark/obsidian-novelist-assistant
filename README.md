# Obsidian Novelist Assistant

面向中文写作者的 Obsidian 插件，辅助小说创作流程。

## 功能

- **排版** — 可自定义首行缩进和行高，优化长文写作体验
- **网格线** — 可调节的网格背景，支持自定义间距、比例、粗细和透明度
- **字数统计** — 在文件管理器中实时显示文件字数
- **设定集** — 快速使用词语创建设定文件，支持快速创建或清除链接

## 技术栈

- [Obsidian API](https://docs.obsidian.md) — 插件框架
- [TypeScript](https://www.typescriptlang.org) — 类型检查
- [Svelte](https://svelte.dev) — 响应式 UI 组件
- [Vite](https://vitejs.dev) — 构建工具
- [Paraglide JS](https://inlang.com/m/gerre34r/library-inlang-paraglideJs) — 类型安全国际化
